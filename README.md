
### Project Description
Project 1 represents a functional Minimum Viable Product (MVP) core application build. This architecture bridges the structural multi-screen mechanics learned in **Lab 4 (Navigation)** with a cohesive set of fully active interactive modules, dynamic entry forms, state validations, and complex screen transitions tailored toward a working user use-case.

### Core Implementation Modules
* **Comprehensive Navigation Network:** Integrated `NavHost` flow linking multiple core layouts seamlessly with active argument data passing.
* **Complex UI Interaction States:** Robust data capturing modules handling multiple entry forms, validation logics, and reactive error processing.
* **Componentized Layout Hierarchies:** Deep structural alignment utilizing reusable layout components, cohesive Material Design 3 elements, and clean view styling.
* **Isolated Architecture (MVVM):** Decoupling presentation layers from business logic rules via state-hoisting and standard ViewModels.

### Built With
* Kotlin & Jetpack Compose Framework
* Jetpack Navigation Components
* Jetpack Lifecycle & ViewModel Architecture
* Android Studio

### Setup & Deployment
1. Clone this repository locally.
2. Open the project folder inside **Android Studio**.
3. Synchronize all necessary Gradle dependencies.
4. Run via an active Android Virtual Device (AVD) emulator or a physical device.
